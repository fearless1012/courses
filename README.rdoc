== Courses

Courses is a web application based on {Rails}[http://rubyonrails.org/] and {Backbone.js}[http://backbonejs.org/] used to facilitate easy access of course information to students (such as, refering to course topics or checking assignments, project deadlines, etc.)

== App Link

The application will be put up at http://courses.nitt.edu
